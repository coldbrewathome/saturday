# FamHop — Intro Video Storyboard & Script

**Product:** FamHop (kids) · famhop.com
**Goal:** In ~30 seconds, show parents that FamHop turns "what do we do this weekend?" into a ready-made plan.
**Tone:** Warm, energetic, reassuring. Brand color = coral. Friendly rounded type, light/white backgrounds.
**Format:** Hero/explainer (~30s) for site + YouTube + landscape social. A 15s vertical cut for IG/TikTok/Shorts follows at the end.
**Music:** Upbeat, light acoustic/indie-pop. Soft "pop" SFX on each card appearance; gentle whoosh on transitions.

> Voiceover (VO) lines are written for a real narrator but also work verbatim as on-screen captions if you go voiceover-free. Every feature named below is real and live in the product.

## Shot-by-shot notes (for the editor)

- **0:00 Cold open** — Real or stock footage of a slightly chaotic-but-loving household reads better than animation here; it earns the relief that comes next. Keep it under 3s.
- **0:03 Logo sting** — This is the emotional turn from "stressed" to "solved." Hard cut to bright white, coral accent, the pin bounce gives it life. Don't linger.
- **0:05 Weekend guide** — Use a screen recording of the live `Weekend (Fri–Sun)` view. Cards animating in sequence (staggered ~120ms) is the hero moment; let it breathe.
- **0:10 Hop Now** — Show the literal **Hop Now picks** with distance labels. The promise is *speed* — keep the tap-to-result feel snappy.
- **0:14 Editor's picks** — Use the real **"Editor's picks — starter plans"** rail, then the map view where saved places order into a plan. This is the "FamHop does the work" beat.
- **0:19 Filters** — Highlight the age-band UI (kids-app only) plus a spot filter chip. Quick — it's a reassurance beat, not a feature dump.
- **0:23 Share/vote** — Use the real "Votes are in on a shared plan" interaction. Family-decision-making is a genuine differentiator; show faces reacting.
- **0:27 CTA** — Bookend with the same family from the cold open, now happy and out. End on a stable logo + URL frame held ~2s so it's screenshot-able.
