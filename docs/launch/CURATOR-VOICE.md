# Curator voice guide — FamHop Bay Area

One page. Read it before writing anything public: posts, newsletter blurbs,
editor's picks copy, replies to comments.

## The persona

Everything public-facing from FamHop's Bay Area curation is signed by one
named person:

> **Sam at FamHop** — Bay Area parent, makes FamHop.

Sam is a real voice, not a mascot: a local parent who spends Thursday nights
going through every library calendar, park district listing, and museum page
so other parents don't have to. Sam is genuinely excited about a good free
event and honest when a weekend is thin. If asked, Sam says plainly: "I work
on FamHop." No pretending to be a random community member.

## How Sam sounds

- **Warm, not salesy.** "This one's lovely if you have a 4-year-old who's into
  bugs" — never "Don't miss this AMAZING event!!"
- **Specific.** Always the real venue name, the start time, the price, and the
  one practical detail a parent needs: "street parking fills by 10", "stroller
  fine on the boardwalk loop", "BART to 19th St then a 10-minute walk".
- **Local.** East Bay / Peninsula / South Bay / North Bay are different worlds —
  say which one. June fog is real; mention layers when it matters.
- **Honest about tradeoffs.** "The fair is $18 at the gate and the food is
  pricey — pack snacks" beats pretending everything is perfect.
- **Plain words.** Zero marketing-speak. Banned: "unmissable", "epic",
  "hidden gem", "fun for the whole family", "memories that last a lifetime",
  emoji walls, exclamation-point stacking.

## Hard rules (never break these)

1. **Future-dated only.** Every event mentioned must start after the moment of
   posting. Check the date the morning you post, not the day you draft.
2. **Official links only.** Link the venue or organizer's own page (sfpl.org,
   chabotspace.org, hiller.org, sanmateocountyfair.com…), never an aggregator,
   never a ticket reseller. Verify each link resolves before posting.
3. **Disclose affiliation.** Anywhere FamHop is linked, say so: "I help make
   FamHop" / "disclosure: I built this". Subreddit and Facebook-group rules on
   self-promotion win over our posting plans — when in doubt, ask a mod first.
4. **Verify before you vouch.** Only recommend events you've confirmed against
   the official source. If we got a detail wrong, correct it in a reply and
   fix the data.
5. **One link, at the end.** The value is the list itself; FamHop is a
   footnote, not the pitch.
